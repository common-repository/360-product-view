== Installation ==

Manual installation is easy and takes fewer than one minute.

1. Download the plugin from wordpress.org, unpack it and upload the <strong>[360 product view]</strong> folder to your <strong>wp-content/plugins/</strong> directory.
2. Activate the plugin through the ‘Plugins‘ menu in WordPress.
3. Go to your main <strong>Production Detail</strong>, and configure the basic options (total frames, frame, display, etc);
4. You need to upload 52 frames of your product to get smooth 360 view.


You’re done. Enjoy.
